# CleanMAC - 依赖与缓存智能清理工具 🚀

> **Vibe Coding 时代的 Mac 磁盘空间拯救神器。**  
> 专门解决因频繁使用 AI 工具进行开发时，系统自动下载海量包与库（如 `node_modules`、`venv`、HuggingFace 模型等）导致 Mac 存储空间被迅速占满的痛点。

---

## ✨ 核心特性

- 🖥️ **Premium 科技感仪表盘**：采用极暗霓虹配色与玻璃态卡片（Glassmorphism）设计，动态渲染 Mac 数据卷的已用空间百分比与可用容量，实时评估可释放空间。
- 🔍 **多语言依赖流式扫描 (SSE)**：
  - 基于 Server-Sent Events (SSE) 协议实现流式加载，在大文件夹扫描时前端列表实时累加，无需漫长等待。
  - 支持智能扫描项目级目录：`node_modules` (Node.js)、`venv` / `.venv` (Python)、`target` (Rust)、`Pods` (CocoaPods) 以及通用的 `build` / `dist` 编译目录。
  - 扫描列表默认按**占用体积降序排序**，并支持切换为**最后修改时间排序**，助您一眼定位“空间大户”。
- ⚡ **全局开发依赖与 AI 缓存一键清理**：
  - 自动获取并列出全局包管理器缓存：`npm`、`yarn`、`pnpm`、`pip`、`cargo`、`go`、`maven`、`gradle`、`cocoapods`、`homebrew` 的占用体积。
  - 特别支持 **HuggingFace AI 模型缓存** 的智能检测与清理。
  - 在计算全局缓存的数秒等待时间内，展示酷炫的**微光扫光骨架屏（Skeleton Loading）**，极具科技体验感。
- 🚄 **高速体积计算与安全防御**：
  - 针对 macOS 文件系统深度优化，利用系统原生 `du -sk` 命令在瞬间算出几百万小文件的巨型依赖目录体积。
  - 内置严格的安全过滤规则，严禁通过 API 清理根目录、系统文件夹、桌面、下载等核心用户目录，杜绝误操作。

## 🛠️ 技术栈

- **前端**：Vite + React + Vanilla CSS (手写暗黑科技风设计系统)
- **后端**：Node.js + Express (本地轻量级服务)

## 🚀 快速开始

### 1. 克隆并进入项目目录
```bash
git clone https://github.com/anzaiyes/cleanMAC.git
cd cleanMAC
```

### 2. 安装依赖
```bash
npm install
```

### 3. 一键启动服务
```bash
npm run dev
```
此命令将利用 `concurrently` 同时在后台运行 Node 后端服务（监听端口 `3005`）与 Vite 前端开发服务器。

### 4. 浏览器访问
在浏览器中打开以下网址即可开始清理：
👉 **[http://localhost:5173](http://localhost:5173)**

## 📸 界面布局

本应用包含三列等高自适应网格设计：
1. **控制中心（左列）**：磁盘空间圆环状态图、扫描路径配置输入。
2. **项目依赖列表（中列）**：支持批量/单个清理，带排序控制与路径自动省略号防溢出机制。
3. **全局开发缓存（右列）**：防止单位折行的两行规整排版，支持一键清空全局备份，找回数十 GB 空间。

## ⚠️ 安全提示

- 本工具在执行“清理”时会永久删除文件且无法通过废纸篓找回。
- 清理项目依赖（如 `node_modules`）后，该项目将暂时无法直接运行，需在项目目录下重新执行 `npm install` 恢复依赖；清理全局缓存（如 `npm` 缓存）不会影响任何现有项目的运行，仅会导致下次安装新包时需要重新联网下载。
- 本地接口已被加上安全边界保护，拒绝接收除规则设定目录以外的删除请求。
